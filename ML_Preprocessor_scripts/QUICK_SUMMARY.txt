❓ ARE WE TRAINING A MODEL?
   ❌ NO

✅ WHAT ARE WE USING?
   • Pretrained Sentence-BERT (SBERT) model: all-MiniLM-L6-v2
   • Source: Hugging Face (Open Source, MIT License)
   • Status: Frozen weights (not modified)

📊 MODEL DETAILS:
   • Name: all-MiniLM-L6-v2
   • Type: Sentence Transformer for semantic embeddings
   • Base: DistilBERT (6 transformer layers)
   • Parameters: 22.7 million
   • Output Dimension: 384 (embedding vector size)
   • Training Data: 215 million sentence pairs (pre-trained)
   • Inference Speed: ~2000 sentences/second
   • Model Size: ~90 MB

🔄 WHAT WE DO:
   1. ✅ Download pretrained model from Hugging Face (first run only)
   2. ✅ Embed your 338 interview questions using the model
   3. ✅ Build FAISS index for fast semantic search
   4. ✅ At runtime, embed user's resume and search for similar questions

❌ WHAT WE DON'T DO:
   • Train the model
   • Modify model weights
   • Fine-tune on your data
   • Use gradient descent or backprop
   • Require GPU for inference

⏱️ PERFORMANCE:
   • Model load time: ~2 seconds (first run only)
   • Embed 338 questions: ~0.5 seconds
   • Search at runtime: ~50ms per query
   • Total latency: ~100ms (fast enough for web)

💰 COST:
   • Model: FREE (open source)
   • Training: NOT NEEDED
   • GPU required: NO (CPU inference is fast enough)
   • Licensing: MIT (commercial use allowed)

🎯 BOTTOM LINE:
   We use a pre-learned semantic embedding model from Hugging Face
   and build an index on top for fast question retrieval.
   Zero training involved. Pure inference! 🚀
